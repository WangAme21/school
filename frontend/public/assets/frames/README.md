# Theme Borders Assets
Place your custom border images or SVGs here.

## How to use:
1. Upload your border image (e.g., `neon-border.png`).
2. Tell me the filename.
3. I will update the CSS to use `url('/assets/frames/your-filename.png')` for the themes.

## Recommended sizes:
- Use transparent PNGs or SVGs.
- Recommended size: 1200x800px or larger for high quality.
